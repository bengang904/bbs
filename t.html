Windows 启用 Telnet 教程

步骤 1： 按下 + R 打开 “运行” 窗口。
步骤 2： 在输入框中键入：cmd
步骤 3： 按下 Ctrl + Shift + Enter， 以管理员身份打开命令提示符。
步骤 4： 在管理员命令提示符中输入以下命令，并按回车：
C:\Users\xxx> dism /online /Enable-Feature /FeatureName:TelnetClient

等待操作完成后，第一次可能会提示重启电脑。

步骤 5： 重启后，打开 cmd，输入命令并按回车：
C:\Users\xxx> telnet 【服务器IP或域名】

即可远程连接到目标服务器。

macOS 启用 Telnet 教程

步骤 1： 按下 ⌘ +       ， 搜索：终端，并打开。也可在应用程序中找到。
步骤 2： 验证 Homebrew 是否安装，若无即会安装。
~ % command -v brew >/dev/null 2>&1 || /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装 telnet

~ % brew install telnet
步骤 3： 安装完成后，在终端输入：
~ % telnet 【服务器IP或域名】

即可远程连接到目标服务器。
